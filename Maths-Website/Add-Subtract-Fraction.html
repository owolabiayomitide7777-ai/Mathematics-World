Lesson Content

Addition and Subtraction of Fractions

A fraction is a part of a whole. It is written in the form a / b where:

  • a is the numerator (top number)

  • b is the denominator (bottom number)

1. Addition of Fractions with the Same Denominator

To add fractions with the same denominator:

  1. Add the numerators.

  2. Keep the same denominator.

  3. Simplify if necessary.

Examples

Example 1: Simplify (3 / 8) + (2 / 8)

Solution

(3 / 8) + ( 2 / 8)

= (3 + 2) / 8

= 5 / 8

Answer: 5 / 8

2. Addition of Fractions with Different Denominators

To add fractions with different denominators:

  1. Find the LCM of the denominators

  2. Convert (remane) the fractions to have the same denominator.

  3. simplify the answer.

Example 2: Evaluate: (1 / 4) + (1 / 6)

Solution

LCM of 4 and 6 = 12

(1 / 4) = (3 / 12) and  (1 / 6) = (2 / 12)

(3 / 12) + (2 / 12)

= 5 / 12

Answer: 5 / 12

Example 3: Evaluate: 3(7 / 20) + 5(1 / 5)

Solution

Step 1: 3 + 5 = 8

Step 2: LCM of 20 and 5 = 20

(7 / 20) = (7 / 20) and  (1 / 5) = (4 / 20)

(7 / 20) + (4 / 20)

= 11 / 20

Combine Step 1 and 2: 8 + 11 / 20

=8(11 / 20)

Answer: 8(11 / 20)

3. Subtraction of Fractions with the Same Denominator

To subtract fractions with the same denominator:

  1. Subtract the numerators.

  2. Keep the same denominator.

  3. Simplify if needed.

Example 4: Subtract (4 / 9) from (7 / 9)

Solution

(7 / 9) - (4 / 9)

= (7 - 4) / 9

= 3 / 9

= 1/3

Answer: 1/3

4. Subtraction of Fractions with Different Denominators

To subtract fractions with different denominators:

  1. Find the LCM of the denominators

  2. Convert (remane) the fractions to have the same denominator.

  3. Subtract the numerators.

  4. simplify the answer.

Example 5: Evaluate: (5 / 6) - (1 / 4)

Solution

LCM of 6 and 4 = 12

(5 / 6) = (10 / 12),   (1 / 4) = (3 / 12)

(10 / 12) - (3 / 12)

= 7 / 12

Answer: 7 / 12

Example 6: Simplify: 3(5 / 8) - 1(3 / 4)

Solution

Step 1: 3 - 1 = 2

Step 2: LCM of 8 and 4 = 8

(5 / 8) = (5 / 8) and  (3 / 4) = (6 / 8)

(5 / 8) - (6 / 8)

Borrow 1 from the whole number

This becomes: ((5 + 8) - 6) / 8

= (13 - 6) / 8

= 7 / 8

Combine Step 1 and 2: 1(7 / 8)

Answer: 1(7 / 8)

Assessment on Addition and Subtraction of Fractions

Section A: Addition of fractions

Simplify the following fractions.

  1. 1/4 + 1/3 + 5/6

  2. 1/3 + 1/4 + 2/5

  3. 2/3 + 1/4 + 5/12

  4. 11/12 + 5/6 + 1/3

  5. 4(2/3) + 3(3/4) + 5(1/6)

  6. 1/8 + 3/32 + 1/2 + 3/4

  7. 5(4/7) + 1/4 + 3(1/2)

  8. 10(1/2) + 7 (1/3) + 3(1/4)

Section B: Subtraction of Fractions

Evaluate the following fractions.

  1. 1/6 - 1/9

  2. 3/8 - 1/3

  3. 7/8 - 3/4

  4. 10(7/10) - (4/5)

  5. 4(11/24) - 2(7/16)

  6. 4(3/8) - 2(2/3)

  7. 7(3/4) - 6(1/2)

  8. 7(3/8) - 3(1/2)
